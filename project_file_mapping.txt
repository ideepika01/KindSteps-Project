KINDSTEPS PROJECT - FILE MAPPING GUIDE
========================================

This document maps each functional page/feature to its corresponding frontend and backend files.

1. AUTHENTICATION
-----------------
Login Page:
- Frontend HTML: frontend/index.html
- Frontend CSS:  frontend/style.css, frontend/styles/globals.css
- Frontend JS:   frontend/js/auth.js, frontend/js/config.js, frontend/js/utils.js
- Backend API:    backend/app/routers/auth.py

Signup Page:
- Frontend HTML: frontend/pages/signup.html
- Frontend CSS:  frontend/styles/signup.css, frontend/styles/globals.css
- Frontend JS:   frontend/js/auth.js, frontend/js/config.js, frontend/js/utils.js
- Backend API:    backend/app/routers/auth.py
